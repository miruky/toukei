政府統計 / e-Stat

日本の統計を、
時系列で読む。

人口・出生・雇用・物価といった主要指標の長期推移を一望し、e-Stat APIのappIdを入れれば任意の統計表をその場で取得して同じ折れ線で描く。値は概数で、正確な数字は各統計の公表資料にあたってほしい。

Indicators

主要指標の推移

同梱スナップショット(公表値を丸めた概数)による長期の動き。

Live API

e-Statから統計表を取得する

e-Stat API の利用登録で得たappIdと、可視化したい統計表ID(statsDataId)を入力する。appIdはこのブラウザのlocalStorageにのみ保存する。

出典: 政府統計の総合窓口(e-Stat)。 ライブ取得はe-Stat APIをブラウザから直接呼び出し、このサイトはデータを保存しない。
